ADHD 플래너 제품의 SNS 브랜딩을 자동화하는 반자동(승인제) 콘텐츠 에이전트.

핵심 아이디어:
- ADHD 플래너의 메시지("흐릿한 목표 → 지금 할 첫 행동")를 인스타그램 + 스레드 콘텐츠로 변환.
- 기획→카피→이미지까지 자동 생성, 사람은 하루 한 번 텔레그램에서 "게시/수정/폐기" 1탭 승인만.
- 자동화 90% / 사람 개입 10%. 완전 무인이 아니라 "5분 승인제 반자동" — 진정성 확보가 핵심 제약.

콘텐츠 기둥 4종: 공감형 / 실전팁형 / 비포애프터형 / 빌드인퍼블릭형.

발행 리듬:
- 인스타: 주 3회(월=공감, 수=실전팁, 금=비포애프터/빌드인퍼블릭 격주), 카드뉴스 5~7장.
- 스레드: 매일 1회 텍스트(월·목 진지 / 화·금 일상+홍보 / 수 질문형 / 토·일 짧은 공감), 280자 내외.

생성 규칙: 추상어 금지·구체 장면, 자책 유발 금지·복귀 함께 제시, 해시태그.

가드레일(생성+승인 양쪽): 미확인 통계 금지, 남의 사연/캡처 금지(익명 재구성), 의학적 단정 금지+면책, 출처 표기.
게시 전 자동 체크리스트 4종: 미확인 통계 / 남의 사연·캡처 / 의학 단정 / 출처 필요 이미지.

기술 스택: Claude API(카피), pencil.dev(이미지 슬롯), 텔레그램 봇(승인), n8n(오케스트레이션) + IG Graph API + Threads API(게시·수집).
구동: Cron(생성을 깨움) + Webhook(게시 버튼/댓글). 생성은 완전 자동, 게시는 사람 1탭 트리거.

MVP 범위: 4기둥 + 2주치 캘린더, IG/스레드 생성 프롬프트(가드레일 포함), pencil.dev 카피 슬롯 구조,
n8n 워크플로우(Cron+Webhook)+텔레그램 승인봇, 게시 후 성과 수집(도달·저장·댓글).
Out of scope: 완전 무인 게시, 유료 광고, 멀티 계정, 댓글 완전 자동 응대.

지표: 도달/노출, 저장·공유율, 팔로워 증가+프로필→랜딩 클릭(핵심), 발행 지속률.
